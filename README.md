# 🗂️ Trello Clone

Um clone do **Trello** para gerenciar tarefas.

---

## ✨ Funcionalidades

- Criar, e excluir tarefas.
- Definir a priodade da Tarefa
- Definir Estado da Tarefa
- Definir Data para a tarefa
- Mover a tarefa de estado.

---

## ⚙️ Tecnologias

- **Frontend:** React + Tailwind CSS
